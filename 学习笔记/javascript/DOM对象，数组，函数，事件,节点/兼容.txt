如果属性在浏览器中不支持，那么这个属性的类型是underfined

================================================================================
innerHTML:
innerText: 获得或设置双标签中的文字,设置标签中的文本内容
textContent:  获得或设置双标签中的文字,设置标签中的文本内容------------> ie8不支持
兼容方法:
if((typeof xxx.textContent)=='underfined'){
    xxx.textContent = x;
}else {
    xxx.innerText = x;
}
