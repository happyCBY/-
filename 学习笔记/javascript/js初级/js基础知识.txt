javascript原本叫liveScript由网景公司的布兰登~艾奇花十天时间发明的

js发明最初的目的是为了解决服务器和用户之间的交互问题
现在js可以做特效，游戏，移动端，服务端

js分为三部分:
1.ECMASciipt: js的标准语法
2.DOM..... Document Object Model 文档对象模型
3.BOM.....Browser Object Model  浏览器对象模型

js是什么？
1.是一门脚本语言
1)编译语言:需要把代码翻译成计算机所认知的二进制语言，才能够执行。
2)脚本语言:就是不用编译器编译可以直接执行的的语言
js是由浏览器中的javascript引擎直接
2.是一门解释性语言
3.是一门弱类型语言
4.是一门基于对象的语言
5.是一门动态类型语言
代码只有执行到这个位置的时候，才知道这个变量到底存储的是什么东西
对象原本没有东西，里面的属性和方法都可以用  对象名.创建的对象或方法名  创建

第一种动态页面:页面由html+css+JS组成
第二种动态页面:向服务器发生请求，服务器那边没有页面，是动态生成的，返回给客户端

js代码的执行机制:
1.同步: 同一时间只能执行一个任务，必须执行完这个任务才能执行下个任务
2.异步: 同一时间可以执行多个任务，在例如在等待定时器的时候可以执行下次工作(在计算机中实际上不存在异步，计算机是使用快速切换进程的方法来实现异步的操作)
3.执行栈: 按顺序执行的代码(注册事件，开启定时器，变量赋值)
4.任务队列: 当执行栈上的代码都执行完毕后才会执行任务队列上的代码(事件处理程序，定时器执行的函数)
5.事件循环: 执行队列里的任务执行完毕之后，会再返回到执行队列里面查看还有没有新的任务出现


js代码可以分三个地方写:
1.在html页面中的script的标签中写script
2.js代码可以在html的标签中写
3.js代码可以在一个js文件中写，然后在html中引用


js语言需要注意的问题:
1.在一对的script的标签中有错误的js代码，那么该错误的代码后面的js不会执行
2.如果第一对的script标签中有错误的js代码,不会影响下一对的script标签中js代码的执行
3.script 中可以写什么内容 type="text-/javascript"是标准写法或者写language="JavaScript"
 但是在现在的html中我们可以不加，因为我们现在的html遵循的是html5规则的标准
4.有可能会出现下边情况，type和language都出现，这样是可以的，是为了浏览器的兼容问题。


js输入函数: prompt("请输入一个数");
js输出函数: document.write(); console.log(); alert();
